Question:- what is the difference between adduser and useradd command in linux
answer- The `adduser` and `useradd` commands in Linux are both used to create new user accounts, but they have some differences in terms of functionality and usage:
- `useradd` is a simple command that creates a new user account with basic settings.
- `adduser` is a more interactive command that provides a wizard-like interface to create user accounts with 
   additional options and configurations.
